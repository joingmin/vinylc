웹접근성 준수사항

기본 사항

해당 프로젝트는 웹접근성 인증마크 획득을 목표로 진행됩니다.

1. alt 값

- 꾸밈 용도로 사용된 이미지의 경우 빈 alt값만 제공한다. Ex) alt=""

2. 레이블 입력

- 입력 서식의 경우 label 또는 title을 사용하여 레이블을 부여한다.

3. 링크 텍스트 입력

  1. 기능이 있는 링크의 경우 적절한 텍스트 정보를 제공한다.
  2. 동일한 링크텍스트를 제공하더라도 그 내용이 다르다면 각각 자세한 정보 텍스트를 추가하여 제공한다. Ex) read more

4. 새 창 알림

- 새창의 경우 target="_blank"와 title="새창" 을 함께 기재

5. 키보드 이용

  1. 키보드로 컨텐츠의 순차적 이동이 되어야 한다.
  2. 키보드로 모든 컨테츠의 접근 및 이용이 가능해야 한다.
